Data Marketplace API

Self-hosted data pipeline & query API

A lightweight service that ingests public data through modular crawlers, normalises it into structured SQLite databases, and exposes clean REST query endpoints. Built to be small, self-hostable, and easy to extend.

Query Syntax

All searches hit POST /v1/collections/<name>/search with a JSON body.

Python · requests 
import requests

url = "http://localhost:5000/v1/collections/_craigslist_cars/search"

payload = {
    "select":      ["*"],
    "filter":      {
        "country":  "United States",
        "price":    {"$gte": 1000, "$lte": 15000},
        "year":     {"$gte": 2010},
        "mileage":  {"$lt":  150000},
    },
    "order_by":    [{"field": "price", "direction": "asc"}],
    "offset":      0,
}

resp = requests.post(url, json=payload)
print(resp.json())

Filter Operators

OperatorMeaningExample
$eq Exact match (shorthand: pass value directly) {"city": "Denver"}
$ne Not equal {"country": {"$ne": "Canada"}}
$gt / $gte Greater than / greater than or equal {"year": {"$gte": 2015}}
$lt / $lte Less than / less than or equal {"price": {"$lte": 10000}}
$like SQL LIKE pattern — use % as wildcard {"title": {"$like": "%toyota%"}}
$in Value must be in the provided list {"state": {"$in": ["Texas", "California"]}}
