_craigslist_realestate

Real estate listings scraped from Craigslist across US & Canadian cities. Records include pricing, bedrooms, bathrooms, square footage when available, listing URL, image, and geo-location data tagged at crawl time. Bedrooms and square footage can be missing for some listings.

Filter Operators
Operator Meaning Example
$gte Greater than or equal to {"price": {"$gte": 0}}
$lte Less than or equal to {"price": {"$lte": 100}}
$gt Strictly greater than {"price": {"$gt": 0}}
$lt Strictly less than {"price": {"$lt": 100}}
$eq Exact match (shorthand: pass the value directly) {"title": "example"}
$ne Not equal to {"title": {"$ne": "example"}}
$like SQL LIKE pattern match — use % as wildcard {"title": {"$like": "%keyword%"}}
$in Value must be in the provided list {"city": {"$in": ["a", "b"]}}
Example Request
Python · requests 
import requests

# Replace these with the credentials provided by your API platform
# (e.g. RapidAPI, your own deployment, etc.)
API_BASE_URL = "https://<your-api-host>"
API_KEY      = "<your-api-key>"

url = f"{API_BASE_URL}/v1/collections/_craigslist_realestate/search"

headers = {
    "Content-Type":  "application/json",
    "X-Api-Key":     API_KEY,  # header name may vary by platform
}

# Structure the POST body however you need — all fields are optional.
# Use "filter" to narrow results, "order_by" to sort, "page_number" to paginate.
payload = {
    "select":      ["*"],
    "filter":      {
        "country": "United States",
        "price": {"$gte": 1000, "$lte": 15000},
        "bedrooms": {"$gte": 0},
        "square_feet": {"$lt": 100},
        "title": {"$like": "%keyword%"},
        "state": {"$in": ["Texas", "California"]},
    },
    "order_by":    [{"field": "price", "direction": "asc"}],
    "page_number": 1,
}

resp = requests.post(url, json=payload, headers=headers)
print(resp.status_code)
print(resp.json())
